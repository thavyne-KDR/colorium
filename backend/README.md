# Colorium – Backend

Backend do projeto **Colorium**, desenvolvido para gerenciamento de autenticação de usuários.
A API oferece funcionalidades de cadastro, login e acesso a rotas protegidas utilizando **JWT**.

---

## Tecnologias Utilizadas

- Node.js
- Fastify
- PostgreSQL
- JWT (JSON Web Token)
- bcrypt
- pg (driver PostgreSQL)

---

## Requisitos

Antes de rodar o projeto, é necessário ter instalado:

- Node.js
- PostgreSQL
- Um gerenciador de banco (pgAdmin)
